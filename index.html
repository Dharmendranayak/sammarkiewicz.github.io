TumblrBareBone url

More about this project

  • html5 Full HTML5 structure - Free of non semantic markup
  • code2 Based upon Tumblr Tags and logic
  • design Logo design

Firmly convinced that Tumblr could be a great tool to create small and medium blog, I used it to create some of mine with this system.
There are a lot of advantages, including an easy customization and a fast development.
Based upon my experience, and with the help of my friend Antoine Brisset, I decided to elaborate a full html5, free of useless classes and seo-friendly Tumblr structure.
It provides you a solid basis to build a clean and semantic Tumblr theme, without any constraint and some useful options.
The project can be downloaded for free on GitHub, and has been already used by some people.

I was comissioned to redesign SEO Footprints, an online app that allows you to find backlinks targets and to generate content.

HeadUp url

More about this project

  • html5 Full HTML5 structure
  • responsive Mobile versions
  • code2 First use of the Tumblr Framework
  • design Logo design and layout

I have always loved words and more specifically their power on people.
On the basis of a personal experience, I decided to create a Tumblog dedicated to motivational quotes.
With this one I try to select the best quotes talking about inspiration or able to lift up your mind.
Technically, it was also an alibi to work with, and to understand, the Tumblr framework.

Icons provided by IcoMoon
Fonts used : Edmond Sans and Lato
This site was built by using SLIM , SASS and by trying to be as close as possible to SMURF and SMACSS principles
